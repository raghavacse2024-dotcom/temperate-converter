# Temperature Converter in Python

This is a simple Python program that converts temperatures between Celsius, Fahrenheit, and Kelvin. The user inputs a temperature value and selects both the current and target units. The program performs the conversion using appropriate formulas and displays the result.

## Features

- Converts between Celsius, Fahrenheit, and Kelvin
- User-friendly command-line interface
- Error handling for invalid input
- Modular code with conversion functions

## Usage

1. **Run the script:**
    ```
    python temperature_converter.py
    ```
2. **Follow prompts:**
    - Enter the temperature value.
    - Enter the current unit (Celsius, Fahrenheit, or Kelvin).
    - Enter the unit to convert to.

3. **View result:**
    - The program will display the converted temperature.

## Example

Temperature Converter
Enter the temperature value to convert: 100
From (Celsius, Fahrenheit, Kelvin): Celsius
To (Celsius, Fahrenheit, Kelvin): Fahrenheit
100 Celsius is 212.00 Fahrenheit.



## Code Structure

- **Conversion Functions:** Handle conversions between each temperature unit.
- **Main Function:** Handles user input, calls conversion functions, and displays results.
